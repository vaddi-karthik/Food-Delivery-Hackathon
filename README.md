# Food Delivery Data Analysis Hackathon

This repository contains my Jupyter Notebook for a hackathon project focused on data integration and analysis of food delivery data.

## Project Overview
The objective of this project is to combine data from multiple sources (CSV, JSON, and SQL) into a single dataset and perform analysis to answer business-related questions.

## Tools & Technologies
- Python
- Pandas
- SQL (SQLite)
- Jupyter Notebook

## Files Included
- Food_Delivery_Hackathon_Analysis.ipynb
- orders.csv
- users.json
- restaurants.sql

## Notes
All analysis is performed using the final merged dataset created through LEFT JOIN operations to ensure all order records are retained.
